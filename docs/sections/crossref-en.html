Data Anonymisation Stepc back to ToC or Class ToC

IRI: https://w3id.org/GDPRov##DataAnonymisationStep

DataAnonymisationStep anonymises data by transforming it from one form to another along the anonymisation chain. Anonymisation can be represented as a spectrum going from raw user data to pseudo-anonymised data that can be de-anonymised by the same agent/organisation to pseudo-anonymous data that cannot be deanonymised internally, but may be done by external agents who have access to other data, and finally to completely anonymised data.
has super-classes
Data Transformation Step c
is in domain of
generatesAnonymisedData op
